# Web App

The web app will support the following features:
- Individual profile displaying most recent (primary) story for a user
  - As minimal content here as possible. The story should be the content.
  - Access to historical index.
- Permanent endpoints for individual stories
- Index of past stories for individual users
  - Scrolling descending timeline
  - Includes most recent (primary) story
  - Index items link to permanent URL for their story
- Homepage of most recent stories for all users
  - Scrolling descending timeline
  - Includes logged in user
- Account authentication
- Basic account management
